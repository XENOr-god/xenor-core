# xenor-core

Deterministic graph propagation and conservative reward routing engine in Rust.

## Overview

`xenor-core` is the protocol-grade core for experimenting with verifiable incentive systems, graph-based propagation, and conservative routing behavior.

## Why this exists

The goal is to build systems where reward movement, propagation logic, and simulation assumptions can be inspected, tested, and reproduced.

## Current scope

- deterministic propagation
- conservative reward routing
- Rust-first protocol experimentation
- reusable foundation for higher-level XENØr simulations
